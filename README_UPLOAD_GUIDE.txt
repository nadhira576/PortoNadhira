NADHIRA PORTFOLIO - GITHUB UPLOAD GUIDE

Files to upload to your GitHub repository root:
- index.html
- style.css
- script.js
- nadhira_cv_latest.pdf
- .nojekyll
- assets/ folder

How to add photos:
1. Open the assets folder.
2. Upload your photos using the exact names in assets/README_FOTO.txt.
3. Example:
   assets/nadhira-profile.jpg
   assets/moure-proof.jpg
   assets/brokobee-proof.jpg
4. Commit changes.
5. Refresh your GitHub Pages website.

If a photo does not appear:
- Check the file name carefully. GitHub Pages is case-sensitive.
- Make sure the extension matches the HTML path, e.g. .jpg vs .png.
- Make sure the image is uploaded inside assets, not in the root folder.
